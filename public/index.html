Contact Messages

POST /contact-messages
Criar uma nova mensagem de contato
Body: { name, email, message }
GET /contact-messages
Listar todas as mensagens de contato (com paginação)
Query Params: page, limit
GET /contact-messages/:id
Buscar uma mensagem de contato específica
PUT /contact-messages/:id
Atualizar uma mensagem de contato
DELETE /contact-messages/:id
Deletar uma mensagem de contato
🏢

Parkings (Estacionamentos)

POST /parkings
Criar um novo estacionamento
Body: { name, address, country, state, city, number, phone, description }
GET /parkings
Listar todos os estacionamentos (com paginação)
GET /parkings/active
Listar apenas estacionamentos ativos
GET /parkings/:id
Buscar um estacionamento específico
PUT /parkings/:id
Atualizar um estacionamento
DELETE /parkings/:id
Deletar um estacionamento
🅿️

Parking Slots (Vagas)

POST /parking-slots
Criar uma nova vaga de estacionamento
Body: { parkingId, number, isAvailable, isActive }
GET /parking-slots
Listar todas as vagas (com paginação)
GET /parking-slots/available
Listar apenas vagas disponíveis
GET /parking-slots/parking/:parkingId
Listar vagas de um estacionamento específico
GET /parking-slots/:id
Buscar uma vaga específica
PUT /parking-slots/:id
Atualizar uma vaga
DELETE /parking-slots/:id
Deletar uma vaga
